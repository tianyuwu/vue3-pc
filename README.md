# A Project build in Vue3+Typescript+Vite

## 安装

```shell
git clone git@github.com:tianyuwu/vue3-pc.git
cd vue3-pc
yarn
```

## 开发环境

```shell
yarn dev
```

## 构建

```shell
yarn build
```
